# FizzBuzz

## 課題内容

FizzBuzz問題

## 動作

１から２０までの数字を表示します。

- 3の倍数 : Fizz
- 5の倍数 : Buzz
- 3と5の両方の倍数 : FizzBuzz
- それ以外 : 数字

## 作成手順

１．　GitHubで新しいリポジトリを作成しました。
２．　fizzbuzz.jsを作成しました。
３．　for文を使用して1から20までを繰り返す処理を書きました。
４．　if文で１５の倍数・５の倍数・３の倍数の順で判定しました。
５．　console.log()で結果を表示するようにしました。
６．　README.mdを作成し、施業内容を記載しました。
７．　GitHubへ登録しました。
